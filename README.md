# douyin_bot

[![https://telegram.me/dyin_bot](https://img.shields.io/badge/%F0%9F%92%AC%20Telegram-DouYin-blue.svg)](https://telegram.me/dyin_bot)

A telegram bot for douyin[抖音]

## Setup

1.  `$ yarn or npm install`
2.  `$ touch .env`
3.  `$ echo TOKEN=YourToken > .env`
4.  `$ yarn start`
